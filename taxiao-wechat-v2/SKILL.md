# 他晓公众号推文模板 Skill V2

## 概述

此 Skill 为「他晓」公众号提供统一的 HTML 推文模板（V2 版本）。相比 V1，新增了**社群邀请区块**。每次生成推文时，自动在正文前后添加品牌化的开头和结尾。

## 资源文件

所有素材位于 `assets/` 目录：

| 文件 | 用途 |
|------|------|
| `avatar.png` | 公众号头像（蓝色底白字"他晓"） |
| `qrcode_gzh.jpg` | 公众号二维码（蓝白商务风） |
| `qrcode_xcx.jpg` | 小程序二维码（彩虹可爱风） |
| `qrcode_wechat.png` | 微信二维码（社群邀请用） |
| `header_deco.png` | 开头装饰图（2K 无水印） |
| `footer_deco.png` | 结尾装饰图（蓝+彩虹融合，2K 无水印） |
| `template.html` | 完整 HTML 模板（含开头+正文占位+结尾） |

## 使用方法

### 1. 读取模板

生成推文前，先读取模板文件：

```
Read assets/template.html
```

### 2. 组装完整推文

将用户内容包裹在模板的 `<!-- ===== ARTICLE CONTENT START ===== -->` 和 `<!-- ===== ARTICLE CONTENT END ===== -->` 之间。

替换以下变量：
- `{{TITLE}}` → 文章标题
- `{{SUBTITLE}}` → 副标题/摘要

### 3. 输出

- 将完整 HTML 保存到用户工作区
- 文件名格式：`{文章主题}_公众号推文.html`
- 同时将 `assets/` 中的图片复制到输出目录的 `assets/` 子文件夹

## 结尾结构（从上到下）

1. **装饰图** `footer_deco.png`
2. **社群邀请卡片** — 蓝紫渐变背景，内含：
   - 标题：「加入他晓交流社群」
   - 说明文字
   - 白色内嵌卡片：添加微信 + 备注「他晓」邀请进群
   - 微信二维码占位（需用户替换）
3. **双二维码并排** — 公众号 + 小程序
4. **版权信息**

## 设计规范

### 配色方案

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色 | `#1976D2` | 品牌蓝，与头像一致 |
| 辅助蓝 | `#2196F3` | 渐变、高亮 |
| 浅蓝背景 | `#E3F2FD` | 卡片、引用框背景 |
| 正文色 | `#1A2332` | 深色文字 |
| 辅助文字 | `#4A5568` | 次要信息 |
| 弱化文字 | `#718096` | 提示、注释 |

### 字体

- 标题：`Noto Serif SC`（衬线，稳重）
- 正文：`Noto Sans SC`（无衬线，易读）
- 代码：`Courier New`

### 重要：内联样式

所有样式使用**内联 style**，确保粘贴到 135 编辑器或微信公众号后台后格式保留。截图占位符使用 `<img>` 标签，可在编辑器中直接点击替换。
