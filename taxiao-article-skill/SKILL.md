# 他晓公众号推文模板 Skill

## 概述

此 Skill 为「他晓」公众号提供统一的 HTML 推文模板。每次生成推文时，自动在正文前后添加品牌化的开头和结尾。

## 资源文件

所有素材位于 `assets/` 目录：

| 文件 | 用途 |
|------|------|
| `avatar.png` | 公众号头像（蓝色底白字"他晓"） |
| `qrcode_gzh.jpg` | 公众号二维码（蓝白商务风） |
| `qrcode_xcx.jpg` | 小程序二维码（彩虹可爱风） |
| `header_deco.jpg` | 开头装饰图（蓝+彩虹融合） |
| `footer_deco.jpg` | 结尾装饰图（蓝+彩虹融合） |

## 使用方法

### 1. 读取模板

生成推文前，先读取模板文件：

```
Read assets/template_header.html   # 开头模板
Read assets/template_footer.html   # 结尾模板
```

### 2. 组装完整推文

将用户内容包裹在开头和结尾之间，生成完整的 HTML 文件：

```
完整推文 = template_header.html + 用户正文内容 + template_footer.html
```

### 3. 输出

- 将完整 HTML 保存到用户工作区
- 文件名格式：`{文章主题}_公众号推文.html`
- 同时将 `assets/` 中的图片复制到输出目录的 `assets/` 子文件夹

## 设计规范

### 配色方案

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色 | `#1976D2` | 品牌蓝，与头像一致 |
| 辅助蓝 | `#2196F3` | 渐变、高亮 |
| 浅蓝背景 | `#E3F2FD` | 卡片、引用框背景 |
| 极浅蓝 | `#F5F9FF` | 页面背景 |
| 彩虹点缀 | 粉/绿/紫/蓝/橙 | 用于小程序区域装饰，呼应小程序风格 |
| 正文色 | `#1A2332` | 深色文字 |
| 辅助文字 | `#4A5568` | 次要信息 |
| 弱化文字 | `#718096` | 提示、注释 |

### 字体

- 标题：`Noto Serif SC`（衬线，稳重）
- 正文：`Noto Sans SC`（无衬线，易读）
- 代码：`Courier New`

### 结尾双二维码布局

公众号二维码（商务蓝白风）和小程序二维码（彩虹可爱风）风格差异大，采用以下策略和谐共存：

1. **左右并排**，各自独立卡片
2. **统一卡片样式**：白色圆角卡片 + 浅阴影，弱化风格差异
3. **各自标注名称**：公众号标注"公众号 · 他晓"，小程序标注"小程序 · 人格派对"
4. **中间分隔**：用一条细竖线或小间距分隔
5. **底部统一**：共用一行引导文字"长按识别二维码关注"

## 模板变量

在 `template_header.html` 中，以下内容需要替换为实际值：

| 变量 | 说明 | 示例 |
|------|------|------|
| `{{TITLE}}` | 文章标题 | MemoryOS - AI 驱动的记忆管理 |
| `{{SUBTITLE}}` | 副标题/摘要 | 用 AI 重新定义你的知识宇宙 |

在 `template_footer.html` 中无需替换，内容固定。
