# 他晓公众号推文模板 Skills

为「他晓」公众号提供统一的 HTML 推文模板，用于 Claude Code 自动生成品牌化的公众号推文。

## 包含的 Skills

### taxiao-article-skill (V1)

基础版公众号推文模板，提供：
- 开头：个人头像 + 公众号名称 + 文章标题区
- 结尾：公众号二维码和小程序二维码左右并排展示 + 关注引导

### taxiao-wechat-v2 (V2)

增强版公众号推文模板，在 V1 基础上新增：
- 社群邀请区块（加微信备注「他晓」进群）
- 微信二维码
- 更丰富的资源文件

## 使用方法

### 安装

将对应的 skill 目录复制到 Claude Code 的 skills 目录：

```bash
# 选择 V1 或 V2 版本
cp -r taxiao-article-skill ~/.claude/skills/
# 或
cp -r taxiao-wechat-v2 ~/.claude/skills/
```

### 触发方式

在 Claude Code 中使用以下关键词触发：
- "生成公众号推文"
- "写公众号文章"
- "创建推文模板"
- "生成 WeChat article"

## 配色方案

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色 | `#1976D2` | 品牌蓝 |
| 辅助蓝 | `#2196F3` | 渐变、高亮 |
| 浅蓝背景 | `#E3F2FD` | 卡片背景 |
| 正文色 | `#1A2332` | 深色文字 |

## 字体

- 标题：`Noto Serif SC`（衬线，稳重）
- 正文：`Noto Sans SC`（无衬线，易读）
- 代码：`Courier New`

## 输出格式

生成的 HTML 文件采用内联样式，可直接粘贴到：
- 135 编辑器
- 微信公众号后台

文件命名格式：`{文章主题}_公众号推文.html`
