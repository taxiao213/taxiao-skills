# 他晓公众号推文模板 Skills

为「他晓」公众号提供统一的 HTML 推文模板，用于 Claude Code 自动生成品牌化的公众号推文。

> **推荐使用 `taxiao-wechat-v3`** — 全内联样式 + HTML align 属性，微信公众号/135 编辑器完美兼容，排版最稳定。

## 包含的 Skills

### taxiao-wechat-v3 (V3) ⭐ 推荐

从已发布文章中提炼的完整写作规范模板，核心改进：
- **全内联样式 + HTML align 属性**，粘贴到 135 编辑器 → 微信公众号后格式完美保留
- 完整的写作风格规范（语气、标题、正文、结构模式）
- 三种文章结构模式：会议纪要型 / 创意延伸型 / 数据科普型
- 丰富的排版组件（callout、信息卡片、链接卡片、链接列表等）
- 微信兼容性最佳：不使用 `<style>` 标签、不使用 flex/grid、table 布局

### taxiao-wechat-v2 (V2)

增强版公众号推文模板，在 V1 基础上新增：
- 社群邀请区块（加微信备注「他晓」进群）
- 微信二维码
- 更丰富的资源文件

### taxiao-article-skill (V1)

基础版公众号推文模板，提供：
- 开头：个人头像 + 公众号名称 + 文章标题区
- 结尾：公众号二维码和小程序二维码左右并排展示 + 关注引导

## 使用方法

### 安装

推荐安装 V3，同时也支持安装旧版本：

```bash
# 推荐：安装 V3
cp -r taxiao-wechat-v3 ~/.claude/skills/

# 或安装旧版本
cp -r taxiao-wechat-v2 ~/.claude/skills/
# 或
cp -r taxiao-article-skill ~/.claude/skills/
```

### 触发方式

在 Claude Code 中使用以下关键词触发：
- "生成公众号推文"
- "写公众号文章"
- "创建推文模板"
- "生成 WeChat article"

## 配色方案

V3 完整配色：

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色 | `#1976D2` | 品牌蓝，标题左边框、强调文字 |
| 深蓝 | `#1565C0` | strong 强调色 |
| 浅蓝背景 | `#E3F2FD` | callout 框背景 |
| 极浅蓝 | `#F0F7FF` | 链接卡片背景 |
| 正文色 | `#1A2332` | 标题文字 |
| 辅助文字 | `#4A5568` | 正文段落 |
| 弱化文字 | `#718096` | 副标题、提示 |
| 边框 | `#E2E8F0` | 卡片边框 |
| 分隔线 | `#BBDEFB` | 淡蓝分隔点 |

## 字体

- 标题：`Georgia, 'Noto Serif SC', serif`
- 正文：系统默认（V3 不设置 font-family，微信会用系统字体）

## 输出格式

生成的 HTML 文件采用内联样式，可直接粘贴到：
- 135 编辑器
- 微信公众号后台

文件命名格式：`{文章主题}_公众号推文.html`
